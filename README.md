# 🎲 RollTogether

<sub>🇧🇷 <a href="./README.pt-BR.md">Readme em Português</a></sub>

### Roll Dice. Create Rooms. Play Together.



A virtual tabletop for RPGs and board games featuring immersive 3D dice rolling, real-time multiplayer synchronization, and modern web technologies.

---

## 🌐 Live Demo

https://paulo-ricardo-ffg.github.io/RollTogether/

---

## ✨ Features

### 🎲 Dice Collection

* D4 (Tetrahedron)
* D6 (Cube)
* D8 (Octahedron)
* D10 (Pentagonal Trapezohedron)
* D12 (Dodecahedron)
* D20 (Icosahedron)
* D100 (Percentile Dice)

### 🌍 Real-Time Multiplayer

* Shared rooms
* Invitation codes
* Synchronized dice rolls
* Roll history
* Firebase Realtime Database
* Fast room sharing

### 🎨 Modern User Experience

* Animated 3D dice
* Responsive design
* Mobile-friendly interface
* Smooth animations
* Fast loading times
* Clean and intuitive UI

## 🚀 Running Locally

### Clone Repository

```bash
git clone https://github.com/paulo-ricardo-ffg/RollTogether.git

cd RollTogether
```

### Python

```bash
python -m http.server 8000
```

### Node.js

```bash
npx live-server
```

Open:

```text
http://localhost:8000
```

## 🎮 How to Play

### Single Player

1. Select a dice
2. Roll the dice
3. View the result

### Multiplayer

1. Create a room
2. Share the room code
3. Invite friends
4. Watch all rolls synchronize in real time

## ⚠️ Security Notice

This project was developed primarily as a learning, experimentation, and rapid prototyping project.

The current version contains known security weaknesses and should not be considered production-ready.

Areas requiring future improvement include:

* Firebase security rules
* Input validation and sanitization
* Authentication and authorization
* Abuse prevention and rate limiting
* Client-side trust assumptions
* Data integrity validation

If you plan to deploy, fork, or use this project in production, a comprehensive security review is strongly recommended.

Security hardening is planned for future releases.
