# 🎲 RollTogether

<sub><a href="./README.md">🇺🇸 Readme in English</a></sub>

### Role Dados. Crie Salas. Jogue Junto.

Uma mesa virtual para RPGs e jogos de tabuleiro com rolagem de dados 3D imersiva, sincronização multiplayer em tempo real e tecnologias web modernas.

---

## 🌐 Demonstração

https://paulo-ricardo-ffg.github.io/RollTogether/

---

## ✨ Recursos

### 🎲 Coleção de Dados

* D4 (Tetraedro)
* D6 (Cubo)
* D8 (Octaedro)
* D10 (Trapezoedro Pentagonal)
* D12 (Dodecaedro)
* D20 (Icosaedro)
* D100 (Dado Percentual)

### 🌍 Multiplayer em Tempo Real

* Salas compartilhadas
* Códigos de convite
* Rolagens sincronizadas
* Histórico de rolagens
* Firebase Realtime Database
* Compartilhamento rápido de salas

### 🎨 Experiência Moderna

* Dados 3D animados
* Design responsivo
* Compatível com dispositivos móveis
* Animações suaves
* Carregamento rápido
* Interface limpa e intuitiva

## 🚀 Executando Localmente

### Clonar o Repositório

```bash
git clone https://github.com/paulo-ricardo-ffg/RollTogether.git

cd RollTogether
```

### Python

```bash
python -m http.server 8000
```

### Node.js

```bash
npx live-server
```

Abra:

```text
http://localhost:8000
```

## 🎮 Como Jogar

### Jogador Único

1. Selecione um dado
2. Role o dado
3. Veja o resultado

### Multiplayer

1. Crie uma sala
2. Compartilhe o código da sala
3. Convide seus amigos
4. Veja todas as rolagens sincronizadas em tempo real

## ⚠️ Aviso de Segurança

Este projeto foi desenvolvido principalmente para aprendizado, experimentação e prototipação rápida.

A versão atual possui vulnerabilidades e limitações de segurança conhecidas e não deve ser considerada pronta para produção.

Áreas que necessitam melhorias futuras:

* Regras de segurança do Firebase
* Validação e sanitização de entradas
* Autenticação e autorização
* Proteção contra abuso e spam
* Redução da confiança no lado cliente
* Validação da integridade dos dados

Caso pretenda utilizar este projeto em produção, recomenda-se realizar uma revisão completa de segurança antes da implantação.

O fortalecimento da segurança está planejado para futuras versões.
